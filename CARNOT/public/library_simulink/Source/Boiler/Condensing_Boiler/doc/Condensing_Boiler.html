Condensing_Boiler
Path: CARNOT/Source/Boiler

Purpose
1 x N node model of a condensing boiler with a lookup table fir the efficiency.

Description
The boiler model is a simple model which considers the heat input into the combustion chamber of the boiler. The model does not consider any limitations from the process or combustion control (e.g. delay times, initial burner control during start up).
On the water side of the furnace, a multinode model is used to model the time dependent conditions. The differential equation for the water temperature

characteristic of a node in the boiler is   

(m*c/N)*dTNode/dt = UA/N*(Tamb-Tnode) + mdot*cf(Tin-Tnode)+Pact/N

the actual power is

Pact = eta_lookup_table(Tm, ctrl) * Pgas * ctrl

the electric power consumption is

Pelec = Pel_lookup_table(ctrl)
