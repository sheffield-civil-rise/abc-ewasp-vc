Collector_unglazed_ISO9806
Path: CARNOT/Source/Solar_Thermal

 

 

 

Purpose
Model for an unglazed solar thermal collectors.

Description

 ISO 9806 model of an unglazed solar thermal collector.

Qdot = A*G*(eta0*(1-bu*vwind) - (b1 + b2*vwind)*(Tm-Tamb)/G)

Qdot = mdot*cp*(Tout-Tin)

Tm = (Tout+Tin)/2

Tout = 2*Tm - Tin                                                                                                                                                                     (1)

CapAbs*dTm/dt = -mdot*cp*(Tout-Tin)/A + (G*eta0*(1-bu*vwind) - (b1 + b2*vwind)*(Tm-Tamb))                     (2)

Static pressure is calculated if necessary, i.e. for thermosyphon systems.

 

Problem of the equation

The ISO 9806 equation violates the energy conservation if the inlet temperature is not constant.
To see this replace Tout in (2) with (1) :

CapAbs*dTm/dt = -mdot*cp*(2*Tm-2*Tin)/A + (G*eta0*(1-bu*vwind) - (b1 + b2*vwind)*(Tm-Tamb))

for simplicity set Tm = Tamb, G = 0 and mdot = 0.1 kg/s :

CapAbs*dTm/dt = -2*mdot*cp*(Tm-Tin)/A

The model takes 2 times the energy from the capacity when inlet temperature varies which results in an invented outlet temperature increase just by reducing the inlet temperature.

IT IS RECOMMENDED TO USE THIS MODEL OLNY FOR STEADY STATE CONDITIONS AT THE INLET.
Use Collector_Unglazed_ISO_2node model for system simulation.
