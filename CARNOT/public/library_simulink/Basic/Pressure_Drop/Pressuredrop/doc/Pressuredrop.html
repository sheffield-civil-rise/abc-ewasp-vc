Pressuredrop
Path: CARNOT/Basic/Pressure_Drop

Purpose
The block calculates the pressure drop in a pipe.

Description
The block calculates the pressure drop in a pipe according to the inputs:

  • mdot : massflow in kg/s
  • velocity [m/s]
  • clq values of enterning flow as a vector
  • con : constant pressure drop coefficient in Pa
  • lin : linear pressure drop coefficient in Pa/(kg/s)
  • con : constant pressure drop coefficient in Pa/(kg/s)²

The equation for the pressure drop is:

      dp = con + mdot*lin + mdot²*qua

 

Characteristics
Direct Feedthrough   Yes
Sample time                Inherited from driving block
Vectorized                   No
