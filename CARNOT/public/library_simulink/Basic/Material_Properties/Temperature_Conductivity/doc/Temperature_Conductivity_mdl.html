Temperatature Conductivity  
Path: CARNOT/Basic/Material_Properties

Purpose
Calculates the temperature conductivity of the specified material.

Description
Calculates the temperature conductivity in [m²/s] according to the inputs.

Inputs:

  • THB : Thermo-Hydraulic Bus

The block calls the function "temperature_conductivity" in the CARNOT Carlib Library where the calculation is carried out.

The material properties functions can also be called directly from the command window by typing

      temperature_conductivity(temperature,pressure, fluidtype,fluidmix)

Instead of the italic variables the actual value of the respective property has to be entered. This is possible for the Carlib functions density, thermal_conductivity, heat_capacity, kinematic_viscosity, prandtl, vapourpressure, saturationtemperature, enthalpy, entropy, evaporation_enthalpy and specific_volume.
For more information about the function type

      help temperature_conductivity

to the workspace as for any other MATLAB function.
