Pressure Drop Pipe Theory
Path: CARNOT/Basic/Pressure_Drop

Purpose
Calculate the pressure drop in a pipe according to theoretical values.

Description

Calculate the pressure drop of a fluid flow in a pipe with a  theoretical approach.
Give the hydraulic * diameter as input.

     D_hydr = 4 * Crosssection-Area / Circumference

Difference in static height between inlet and outlet is also taken into account.
Outlet above inlet has a positive height and a decrease in static pressure.

Laminar flow is assumed for Reynolds < 2050. In ths case the friction coefficient is

     fh = 64.0/Re

For Renolds number between 2050 and 3000 a transition zone is considered:
The friction coefficient is a linear interpolation between laminar and tubulent value.

For turbulent flow the coefficient is calculated with the Blasius and Nikuradse correlation.

A developing flow correction according to Lencastre is applied.
