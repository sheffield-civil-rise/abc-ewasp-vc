Pressure Drop static Height
Path: CARNOT/Basic/Pressure_Drop

Purpose
Calculate the static pressure drop or increas due to a difference in static height.

Description

Calculate pressure drop or increase due to statig height.
Calculation is done for Flow_ID > 20000 (see Pump_Main for details)

     dp = height/Nnodes*grav*density(Tnodes)

The length of Tnodes vector defines the number of temperature nodes.

The height is between inlet and outlet. It is divided by the number of nodes Nnodes in the temperature vector "Tnodes".

Height is positive for outlet above inlet when pressure decreases.

 

Inputs

Tnodes : Vector with the nodes temperature in °C 

THB

height : difference of height between inlet and outlet in m (positive for outlet above inlet)

Outputs

THB

 

Parameters and Dialog Box

 

Characteristics

Sample time                Inherited from driving block
Vectorized                   No
