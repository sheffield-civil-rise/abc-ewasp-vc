fitSineToTamb - fits a sine curve to Tamb for ISO 13370 ground model

Function Call

[ok, param] = CarnotCallbacks_StorageConf(fname, bhandle, storagetype)

Inputs

filename - [optional] filename of the weather data file (
bhandle - block handle (result of matlab function gcbh)
storagetype  - 0: buffer storage with inlet and outlet connection
             - 1: buffer storage with charging and discharging pipes

Outputs

Tavg     - average ambient outdoor temperature in °C
ampT     - Amplitude of temperature in K

Description

Fit a sine curve to the annual ambient temperature curve.
T(t) = Tavg - ampT * cos( 2*pi/(365*24*3600) * (t - tshift) )
