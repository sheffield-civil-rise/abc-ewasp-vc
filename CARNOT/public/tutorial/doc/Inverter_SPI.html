Inverter_SPI
Path: CARNOT/Source/Electric_Generator



Purpose
Hybrid Inverter with DC Battery

Description
This model is a simulink conversion of PermodDC Model by HTW Berlin.

Losses are calculated by one quadratic loss equations fro each power path. Simple control dynamics are implemented as PT1 behavior.
Note that this is a discrete model, rate transitions will be necessary when combined with continuous models. The sample time is a user parameter. 1sec is default but values larger than 10s are not recommended for means of accuracy
Detailed documentation of the algorithm can be found here.( https://pvspeicher.htw-berlin.de/wp-content/uploads/PerMod_Documentation.pdf )

Changes to the original model include:

  • no stationary control deviation
  • dead time affects power calculation only (soc of carnot model lags one time step
  • Grid power target input instead of "0"
  • charging from Grid: works but is NOT VALIDATED

Inputs:

  • PV DC-Power
  • Setpoint for maximum AC power output

Outputs:

  • AC Power
  • Inverter Data Bus
