Function Call

ttt = SearchWriteParams(modelname,fid)

Inputs

modelname - name of the Simlink model (typically set to bdroot)
fid - file identifier for the [modelname '_params.txt'] file

Outputs

namelist - character strings of the filenames

Description

This function searches the Simulink model modelname for parameters in masks, constant, FromFile and FromWorkspace blocks. It writes the information in a structured way into the file wit id fid. This function is an example of recursive programming as it calls itself in its definition
