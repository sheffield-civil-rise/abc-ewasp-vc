Function Call

vp = vapourpressure(t, ft, fm)

Inputs

(scalar or vector)
t    :   temperature [°C]
ft   :   fluid_ID (see link to FluidEnum below)
fm   :   fluid_mix

Outputs

vp  : vapour pressure of the fluid in Pa

Description

The function calls the function "vapourpressure" in the CARNOT library Carlib.
The Carlib function is also used by the "Vapourpressure"-block in CARNOT library.

References and Literature

Function is used by: Carnot blocks in basic/material_properties
See also Carnot_Fluid_Types, FluidEnum, Carlib,
 density, enthalpy, entropy, heat_capacity,
 kinematic_viscosity, saturationtemperature, specific_volume,
 temperature_conductivity, thermal_conductivity
